# PMAIDE Practice Quiz

**Use it here: https://rabbicse.github.io/pmaide-quiz/**

Practice app for the University of Dhaka CSE **PMAIDE** admission test (50 MCQs · 100 marks · 60 min · no negative marking).

- **449 curated questions** (including the official 50-question sample paper) — every answer explained
- **123 generator templates** producing unlimited fresh questions with computed answers — code traces, heap/BST drills, shortest paths, SQL results, probability, IQ puzzles
- **Topic drills** with instant feedback (wrong answers come back first) and **timed 60-minute mock exams** mirroring the real paper
- **Status dashboard**: estimated exam score, per-topic accuracy, weak-topic flags, mock history

One self-contained HTML file — works offline once loaded, no account, no server. Progress is stored in your browser's localStorage (per device).

Study notes and the full prep kit live in a separate (private) repository; this repo exists to serve the app via GitHub Pages.
